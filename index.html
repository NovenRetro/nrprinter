NR Printer · Game Boy Printer para ESP32

Firmware v1.0

Instalador web para convertir un ESP32 en una impresora compatible con Game Boy Camera/Game Boy Printer, con galería local, guardado interno e impresión automática en térmicas SC03h compatibles.

Usá Google Chrome o Microsoft Edge desde una PC. Conectá el ESP32 por USB, elegí el puerto COM correcto y presioná instalar. Si la placa no entra en modo flash automáticamente, mantené presionado BOOT durante el inicio del proceso.
Tu navegador no soporta Web Serial. Usá Google Chrome o Microsoft Edge desde una PC. Web Serial requiere HTTPS o localhost.

El proceso carga bootloader, particiones y firmware principal. Al finalizar, el ESP32 inicia en Modo Printer, con WiFi apagado.

¿Qué hace?

  • El ESP32 se comporta como una Game Boy Printer frente a la Game Boy.
  • Guarda las capturas en memoria interna usando LittleFS.
  • Si Auto-print está activo, busca una impresora SC03h compatible e imprime.
  • Si la térmica no está disponible, la captura queda guardada igual.
  • BOOT 2 segundos abre la red local NRPrinter para galería/configuración.

Compatibilidad

Pensado para impresoras tipo SC03h/iPrint con BLE AE30 y TX AE01. No es compatible con cualquier impresora térmica Bluetooth genérica.

Comportamiento de NRPrinter

  • Modo Printer listo: LED apagado.
  • Modo WiFi/Web: LED encendido fijo.
  • Buscando/vinculando SC03h: pulso lento.
  • Imprimiendo en térmica: parpadeo durante el envío.
  • BOOT corto: alterna Auto-print. 4 parpadeos = ON, 2 parpadeos = OFF.
  • BOOT 2 segundos: abre la galería web local.
